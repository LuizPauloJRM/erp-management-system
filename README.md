# erp-management-system
O erp-management-system é um sistema ERP (Enterprise Resource Planning) desenvolvido com foco em aprendizado prático e evolução contínua, simulando cenários reais de aplicações corporativas.  O projeto tem como objetivo central a construção de uma aplicação robusta e escalável, aplicando boas práticas de desenvolvimento backend e frontend com Java, utilizando tecnologias amplamente adotadas no mercado como JSF, JPA e Hibernate.  A aplicação será estruturada em camadas, seguindo padrões de arquitetura utilizados em sistemas empresariais, permitindo fácil manutenção, reutilização de código e crescimento progressivo.
